Notable Computer Science Cources

Taught by Ryan Nutt, I was first introduced into programming in this class and learned all about the basics of Java. From the basics, such as arrays, loops, and methods, to even the more complex, such as inheritance, classes, and sorting, Mr. Nutt guided me from the start and set up a strong foundation for my programming skills.

Taught by Paul Stroud, I began learning more complex data strutures and algorithms in computer science, from queues, heaps, stacks, maps, sets, and trees to binary search and self balancing trees, such as red black trees and AVL trees. Moreover, CS3 was when I began to gain more practical project experience as I worked on 3 different projects throughout the year: I created the card game Love Letter, was the top team member for the Azul Board Game project, and became the leader for the Kingdom Builder project. Overall, not only were my project development skills strengthened, but also my leadership skills in such projects were developed.

Taught by Ryan Nutt, I am able to freely experiment with various topics in computer science while learning about new topics. This class is especially unique in that I am finally able to freely apply all of my knowledge in computer science into projects that are interesting. Specifically, some of my favorite projects from this class include the flashcard project, in which I was able to address an actual need of students, the Connect 4 Project, where I learned about how I can use python to create UIs, and the portfolio project, in which I can not only further develop my web design skills, but also build a timeless portfolio that I can use in the future when searching for jobs. Overall, this class does well in broadening my knowledge on various languages and topics in computer science instead of focusing on just one language.
